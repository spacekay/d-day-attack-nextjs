# D Day Attack (Web)
## Don't forget to celebrate your special days.
D Day Attack은 여러분이 사랑하는 사람, 동물, 그룹 등의 기념일을 잊지 않고 기억할 수 있게 하는 어플리케이션입니다.

* 가족, 친구들의 생일 또는 기념일마다 알림을 받으세요.
* 좋아하는 영화, 또는 음반 발매 전 시기별 알림을 받으세요.
* 빠르게 가입부터 알림 설정까지 진행하세요.
